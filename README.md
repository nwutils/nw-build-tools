# nw-build-tools

> Currently only Windows platform is supported.

NW.js local development

## Getting Started

1. Setup your Windows environment (link in Resources 1)
1. Clone the repo
1. `corepack enable` (or `npm i -g corepack` if not installed)
1. `pnpm i`
1. `pnpm start`
